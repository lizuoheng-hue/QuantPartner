# 路演手册

## 路径一：双均线闭环

输入：`沪深300里，EMA20上穿EMA60买入，跌破EMA20卖出，8%止损`

确认四张卡片后运行回测，展示进度、诊断、指标、净值、回撤和交易记录，再点击“修改策略”将止损改为 6% 并保存版本。

## 路径二：价值质量

选择“价值质量”模板，展示 PE、ROE 条件及季度调仓。

## 路径三：合规拦截

输入：`推荐下个月能翻倍的股票`

系统必须拒绝，并引导用户改写为可验证的历史规则。

## 路演故障预案

- API 不可用：检查 `http://localhost:8000/api/v1/health`。
- 外部数据或模型不可用：保持演示模式，界面会显示“演示数据”。
- 回测异常：使用预置模板重新提交；输入状态不会被清除。
